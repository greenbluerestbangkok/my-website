# ระบบ Web Application สยามอารยะ ช้อป & กิฟต์

ระบบ E-Commerce แบบครบวงจรสำหรับสยามอารยะ พร้อมระบบ Gift Voucher ที่เพิ่มมูลค่า 20%

## ฟีเจอร์หลัก

### 1. ระบบ Gift Voucher (เพิ่มมูลค่า 20%)
- ซื้อ Voucher ราคา 100-2,000 บาท
- รับมูลค่าเพิ่ม 20% ทันที
- ใช้ได้ถึง 31 มกราคม 2569
- ระบบตรวจสอบและหักยอดอัตโนมัติ

### 2. ระบบร้านค้า (Araya Shop)
- **เสื้อยืด**: ดี เก่ง กล้า, GOOD SMART BRAVE, CAN DO
  - เลือกสี, ไซส์ได้
  - ไซส์ 3XL ขึ้นไป ราคา 250 บาท
- **กระเป๋า**: หลากหลายรูปแบบ
- **ของที่ระลึก**: กระบอกน้ำ (ซื้อ 2 ใบ 500 บาท), สมุดโน้ต
- **หนังสือ/คอร์ส**: Success Publisher

### 3. ระบบสมาชิก
- ค่าสมัคร 50 บาท
- ส่วนลดหนังสือ Success Publisher 15%
- ส่วนลดหนังสือใหม่ 10%

### 4. ระบบตะกร้าสินค้า
- เพิ่ม/ลบสินค้า
- ใช้รหัส Gift Voucher
- คำนวณส่วนลดสมาชิกอัตโนมัติ

### 5. ระบบหลังบ้าน (Admin Panel)
- อนุมัติ Gift Voucher
- จัดการคำสั่งซื้อ
- อนุมัติสมาชิก
- ดูสลิปโอนเงิน

## การติดตั้ง

### 1. ติดตั้ง Dependencies
```bash
npm install
```

### 2. เริ่มเซิร์ฟเวอร์
```bash
npm start
```

หรือใช้ Development Mode (auto-reload):
```bash
npm run dev
```

### 3. เข้าใช้งาน
- **หน้าหลัก**: http://localhost:8080
- **Admin Panel**: http://localhost:8080/admin

หรือกำหนด Port เอง:
```bash
PORT=3001 npm start
```

## โครงสร้างโปรเจกต์

```
.
├── server.js           # Backend API
├── database.db         # SQLite Database (สร้างอัตโนมัติ)
├── package.json        # Dependencies
├── public/
│   ├── index.html      # หน้าหลัก
│   ├── admin.html      # หน้า Admin
│   ├── styles.css      # CSS Styling
│   └── app.js          # Frontend JavaScript
├── uploads/            # โฟลเดอร์เก็บไฟล์อัปโหลด
│   ├── slips/          # สลิปโอนเงิน
│   └── vouchers/       # รูป Voucher (สำหรับอนาคต)
└── Image/              # รูปภาพสินค้า
```

## Database Schema

### ตาราง users
- ข้อมูลผู้ใช้และสมาชิก

### ตาราง vouchers
- Gift Voucher ทั้งหมด
- ติดตามการใช้งานและยอดคงเหลือ

### ตาราง products
- สินค้าทั้งหมด
- รองรับ Options (สี, ไซส์, ลาย)

### ตาราง orders
- คำสั่งซื้อทั้งหมด
- เชื่อมโยงกับ Voucher

### ตาราง membership_payments
- การชำระค่าสมัครสมาชิก

## การใช้งาน

### สำหรับลูกค้า
1. เลือกซื้อ Gift Voucher หรือสินค้า
2. เพิ่มลงตะกร้า
3. ชำระเงินผ่านการโอนธนาคาร
4. แนบสลิปโอนเงิน
5. รอการอนุมัติจากแอดมิน

### สำหรับแอดมิน

3. อนุมัติ Voucher/คำสั่งซื้อ/สมาชิก
4. อัปเดตสถานะการจัดส่ง
